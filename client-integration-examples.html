Client Integration Scripts

Option 1: Basic Integration (Recommended)

Homepage/General Pages Script:


<script>
// Avenix Homepage Pixel - Place before closing </body> tag
(function() {
  var pixel = new Image();
  pixel.src = 'https://avenix-pixel.vercel.app/pixel/home?url=' + 
    encodeURIComponent(window.location.href) + 
    '&time=' + Date.now() + 
    '&device=' + encodeURIComponent(navigator.userAgent);
  pixel.style.display = 'none';
  document.body.appendChild(pixel);
})();
</script>

Option 3: Smart Single Script (Advanced)


<script>
// Avenix Smart Pixel - Automatically detects page type
(function() {
  // Configure conversion page detection (clients can customize)
  var conversionPages = ['/thank-you', '/success', '/complete', '/order-complete'];
  var currentPath = window.location.pathname.toLowerCase();
  
  // Determine pixel type
  var pixelType = 'home';
  for (var i = 0; i < conversionPages.length; i++) {
    if (currentPath.includes(conversionPages[i])) {
      pixelType = 'conversion';
      break;
    }
  }
  
  // Send pixel
  var pixel = new Image();
  pixel.src = 'https://avenix-pixel.vercel.app/pixel/' + pixelType + 
    '?url=' + encodeURIComponent(window.location.href) + 
    '&time=' + Date.now() + 
    '&device=' + encodeURIComponent(navigator.userAgent);
  pixel.style.display = 'none';
  document.body.appendChild(pixel);
})();
</script>

Integration Instructions for Clients

  1. Choose an option above (Option 1 recommended for simplicity)
  2. Copy the script(s) you need
  3. Paste before the closing </body> tag on your website
  4. For Option 2: Replace "CLIENT_ID" with your assigned client ID
  5. Test: Visit your pages and verify tracking in your analytics

What Gets Tracked

  • ✅ Page URL
  • ✅ Visit timestamp
  • ✅ Device/browser information
  • ✅ Client identification (domain or ID)
  • ✅ Page type (homepage vs conversion)
  • ✅ IP address (server-side)
